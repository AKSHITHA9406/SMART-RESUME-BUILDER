# SmartResume+

A Java Swing-based Resume Builder application that allows users to create, preview, and export resumes.

## Features
- Multi-tab GUI (Personal Info, Education, Experience, Projects, Skills)
- Live resume preview using Graphics2D
- Resume completeness score
- Job description matching score
- Export resume as image
- Clean and modular OOP design

## Technologies Used
- Java
- Swing & AWT
- Object-Oriented Programming
- Graphics2D

## How to Run
1. Open the project in IntelliJ IDEA
2. Run `SmartResumeApp.java`
3. Fill details and generate preview

## Future Improvements
- PDF export
